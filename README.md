### EcoTrip – Calculadora de Impacto Ambiental para Viagens
Este projeto foi desenvolvido como parte do desafio “Calculadora EcoTrip: Simulador de Impacto Ambiental para Viagens”. 
O objetivo é permitir que o usuário simule o impacto ambiental de uma viagem com base em distância, modo de transporte, número de passageiros e tipo de combustível. 
O sistema calcula as emissões de CO₂ e apresenta equivalentes ecológicos e recomendações gerais.

### Funcionalidades
- Cálculo de emissões totais de CO₂.
- Cálculo de emissões por passageiro.
- Comparação com equivalentes ecológicos (árvores e km de carro).
- Recomendações sustentáveis geradas por lógica interna.
- Interface simples e responsiva.
- Backend em Node.js e frontend em React.
- Comunicação via API REST.

### Tecnologias Utilizadas
Backend
- Node.js
- Express
- CORS

### Frontend
- React
- Vite
- Axios

### NOTA
- O projeto funciona sem integração com IA externa.
- Caso queira integrar IA, configure manualmente no backend e utilize variáveis de ambiente locais.
